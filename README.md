

# AI Email Assistant Extension

An AI-powered browser extension that helps users generate, reply, summarize, and improve emails directly inside their email client using AI.

---

# Features

- AI-generated email replies
- Email summarization
- Professional tone improvement
- Grammar correction
- Smart response suggestions
- Fast and lightweight extension

---

# Tech Stack

## Frontend
- JavaScript
- HTML
- CSS
- Chrome Extension APIs

## Backend
- Java
- Spring Boot
- Spring AI
- REST APIs

## AI
- OpenAI API
